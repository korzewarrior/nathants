system components

Lambda Architecture

Minimal lambda zip containing only 3 files: favicon.png, index.html.gz, and the go binary main. Efficient and lightweight deployment.

Frontend + Backend

Go lambda backend with ClojureScript and React frontend. Full-stack capabilities with seamless integration.

State Management

S3 and DynamoDB for robust state management. HTTP and WebSocket APIs for real-time communication.

DevOps Automation

Streamlined development workflow with automated deployment and testing. Low latency logging for debugging.

quickstart commands

terminal commands:
git clone https://github.com/nathants/aws-gocljs
cd aws-gocljs
cp env.sh.template env.sh # update values

# development and deployment
bash bin/check.sh         # lint
bash bin/preview.sh       # preview changes to aws infra
bash bin/ensure.sh        # ensure aws infra
bash bin/dev.sh           # iterate on backend and frontend
bash bin/logs.sh          # tail the logs
bash bin/delete.sh        # delete aws infra

# with limited bandwidth
bash bin/dev_frontend.sh  # iterate on localhost frontend
bash bin/relay.sh         # iterate on backend via ec2 relay

dependencies

System Requirements

Use the included Dockerfile or install the following dependencies: npm, jdk, go, bash, entr, libaws

AWS Prerequisites

AWS Route53 has the domain or its parent from env.sh. AWS ACM has a wildcard cert for the domain or its parent from env.sh.
